# name

gendd: Generate Dummy Date

## Overview

テストなどで利用する日付データを生成します。

## Requirement

- node.js
- TypeScript
- React
- Jest

## Usage

「生成」ボタンをクリックすると、生成された日時データが表示されます。表示された日時データを選択、コピーすることで、テストなどに利用できます。

## Features

- テストなどに利用するための日時データを生成します。日時データのフォーマットは

> YYYY/MM/DD hh:mm:ss
